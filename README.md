## About

I design and build systems end-to-end.

From low-level **C++ game engines** to full-stack **SaaS platforms** — I care about the whole stack: performance, architecture, and the kind of code that survives scale.

My current obsession is the integration of **local LLMs** into real products: privacy-first, latency-aware, and genuinely useful.

I don't chase frameworks. I chase **correctness**.
